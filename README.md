# HMEQ Loan Default Data Analysis

## Project Overview

This project performs Exploratory Data Analysis (EDA) on the HMEQ (Home Equity) Loan Default dataset using Python. The objective is to analyze customer loan information, identify data quality issues, explore relationships between variables, and discover patterns associated with loan defaults.

---

## Objectives

- Load and inspect the dataset
- Explore data structure and data types
- Handle missing values
- Generate descriptive statistics
- Detect outliers
- Analyze feature distributions
- Visualize relationships between variables
- Identify patterns related to loan default
